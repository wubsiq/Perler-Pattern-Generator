# 像素化图片生成器

## 功能特点

- 🎨 **像素化处理**：将图片转换为像素艺术风格
- 🧩 **拼豆模式**：生成拼豆图纸，支持多种颜色套装
- 📊 **颜色统计**：自动统计每种颜色的拼豆数量及总数量
- 📏 **灵活调整**：可自定义像素块大小、目标尺寸
- 💾 **一键下载**：支持下载像素化图片和拼豆图纸

## 颜色套装

- **Mard 291色**：默认，包含291种丰富颜色
- **Mard 48色**：精简版48色套装
- **Mard 24色**：基础24色套装
- **Perler 24/48/72色**：经典Perler拼豆颜色
- **Hama 10色**：基础10色套装

## 快速启动

### 方法1：使用bat文件（推荐）

双击运行 `start_server.bat` 文件，服务器将自动启动在 http://localhost:8000

### 方法2：手动启动

打开命令提示符，进入项目目录，运行以下命令：

```bash
python -m http.server 8000
```

然后在浏览器中访问 http://localhost:8000

## 使用说明

1. **上传图片**：拖拽图片到上传区域或点击上传按钮
2. **调整参数**：设置像素块大小、目标尺寸等
3. **预览效果**：查看原始图片、像素化结果
4. **拼豆模式**：勾选"拼豆模式"，点击"渲染拼豆图纸"生成拼豆图纸
5. **下载保存**：点击下载按钮保存生成的图片或拼豆图纸

## 技术说明

### 模块划分

- `colors.js`：颜色集合数据
- `colorUtils.js`：颜色处理工具函数
- `app.js`：主应用逻辑
- `index.html`：页面结构
- `style.css`：样式文件

### 颜色匹配算法

采用RGB→XYZ→LAB色彩空间转换，使用CIE76标准计算颜色距离，实现更符合人眼感知的颜色匹配。

## 系统要求

- Python 3.x（用于启动本地服务器）
- 现代浏览器（Chrome、Firefox、Edge等）
- 支持HTML5 Canvas API
